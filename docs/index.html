앱 소개

어떤 앱인가요?

불타는 내인생은 하루의 남은 시간을 양피지가 타들어가는 시각적 효과로 보여주는 카운트다운 앱입니다. 기상 시간부터 취침 시간까지, 시간이 얼마나 남았는지 직관적으로 확인하고 하루를 더 의미있게 보낼 수 있도록 도와줍니다.

일간 카운트다운 주간 / 월간 / 연간 홈 화면 위젯 Dynamic Island 잠금화면 위젯

사용 방법

기상 / 취침 시간 설정

오른쪽 상단 ⚙️ 버튼을 눌러 기상 시간과 취침 시간을 설정하세요. 기상 시간이 되면 양피지가 타들어가기 시작하고, 취침 시간이 되면 카운트다운이 종료됩니다.

기간 선택

상단 탭에서 1일 · 1주 · 1달 · 1년 중 원하는 기간을 선택하면 해당 기간의 남은 시간을 양피지 연소로 표현합니다.

홈 화면 위젯

홈 화면 길게 누르기 → 위젯 추가 → 불타는 내인생을 검색해 일간 / 주간 / 월간 / 연간 위젯을 추가할 수 있습니다. 소형(Small)·중형(Medium)·대형(Large) 크기를 지원합니다.

Dynamic Island · 잠금화면

카운트다운이 시작되면 Dynamic Island와 잠금화면에서 실시간으로 남은 시간을 확인할 수 있습니다. Dynamic Island를 아래로 당기면 양피지 연소 애니메이션과 함께 상세 정보가 표시됩니다.

자주 묻는 질문

위젯과 앱의 시간이 다르게 표시돼요

홈 화면 위젯은 iOS의 구조적 제한으로 실시간 갱신이 불가능합니다. 앱을 열면 위젯 데이터가 자동으로 동기화됩니다. 실시간 카운트다운은 Dynamic Island와 잠금화면 위젯에서 확인하세요.

수면 중에는 어떻게 표시되나요?

취침 시간이 지나면 불타는 하트 애니메이션이 표시되며, 기상 시간까지의 수면 진행률을 보여줍니다. 기상 30분 전부터는 새 양피지가 준비됩니다.

알림은 언제 오나요?

취침 1시간 전에 알림이 발송됩니다. 알림이 오지 않는다면 설정 → 알림 → 불타는 내인생에서 알림이 허용되어 있는지 확인해주세요.

앱이 배터리를 많이 소모하나요?

앱 내 카운트다운은 1초 타이머를 사용하지만, 위젯은 시스템이 관리하는 저전력 타임라인 방식으로 동작해 배터리 영향을 최소화합니다.
