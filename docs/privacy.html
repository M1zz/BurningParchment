요약

불타는 내인생은 개인정보를 수집하지 않습니다

앱이 동작하는 데 필요한 모든 데이터는 사용자의 기기 내에서만 처리되며, 외부 서버로 전송되거나 제3자와 공유되지 않습니다.

🔒 인터넷 연결 불필요 · 수집 데이터 없음

수집 및 저장 정보

기기에 저장되는 정보

앱은 아래 항목만 사용자의 기기 내부(UserDefaults)에 저장합니다.

  • 기상 시간 (시, 분)
  • 취침 시간 (시, 분)

이 값들은 앱과 홈 화면 위젯 간 데이터 공유를 위해 기기 내 App Group 저장소를 통해서만 사용됩니다.

수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 신원 정보
  • 위치 정보
  • 연락처, 사진, 건강 데이터
  • 기기 식별자 (IDFA, IDFV 등)
  • 사용 행태 분석 데이터
  • 광고 관련 데이터

제3자 공유

데이터를 공유하지 않습니다

앱은 어떠한 분석 SDK, 광고 네트워크, 제3자 서비스도 사용하지 않습니다. 사용자 데이터는 외부로 전송되지 않습니다.

알림

로컬 알림만 사용합니다

취침 1시간 전 알림은 기기 내에서 예약되는 로컬 알림으로, 외부 서버를 거치지 않습니다. 알림 권한은 사용자가 언제든지 iOS 설정에서 변경할 수 있습니다.

아동 개인정보 보호

만 14세 미만 아동

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

방침 변경

변경 시 이 페이지를 통해 안내합니다

개인정보 처리방침이 변경될 경우 이 페이지의 최종 업데이트 날짜를 수정하고 내용을 업데이트합니다. 중요한 변경이 있을 경우 앱 업데이트 노트를 통해 별도로 안내할 수 있습니다.
